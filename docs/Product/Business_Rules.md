# Business Rules

## Lead Rules

- A lead is created upon submission of the registration form on the Landing Page.
- A lead must provide, at minimum, an email address and required personalization inputs.

## Email Rules

- One email address can generate only one final song.
- Email uniqueness is validated before personalization and lyrics generation begin.
- A duplicate email attempting to register again is rejected.

## Attempts Rules

- Each user has five lyric attempts.
- Attempts are consumed only by:
  - moderation rejection
  - lyric regeneration (user requests new lyrics)
- Attempts are never consumed during audio generation.
- Once all five attempts are exhausted without an accepted set of lyrics, the user cannot obtain a song.

## Moderation Rules

- All personalization input is checked by Claude moderation before lyrics are generated.
- A moderation rejection consumes one lyric attempt.
- Content that fails moderation must be revised by the user before resubmission.

## Lyrics Rules

- Lyrics are generated by Claude based on validated, moderated personalization input.
- Generated lyrics are shown to the user as a preview before any song generation occurs.
- The user must explicitly accept lyrics before song generation proceeds.
- Requesting new lyrics (regeneration) consumes one attempt.

## Song Rules

- Only one final song is generated per user (per email).
- Song generation only begins after the user accepts a set of lyrics.
- Song generation uses Suno and never consumes a lyric attempt, regardless of success or failure.

## Mood Rules

- The campaign provides exactly four predefined moods.
- Each mood maps to a fixed Suno prompt.
- Users select one mood per personalization; moods are not combined or customized.

## Admin Rules

- The admin panel allows monitoring of leads and campaign submissions.
- The admin panel supports CSV export of lead/campaign data.
- Version 1 supports a single administrator (see BACKLOG_V2.md for multiple administrators).

## Campaign Rules

- Campaign duration is one month.
- The campaign supports generation of up to 3,000 songs in total.
- Only one email is sent per user, containing the final generated song.
- The campaign is temporary and is not intended to operate beyond its defined duration or song cap.
